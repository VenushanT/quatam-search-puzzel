Visualize how different algorithms solve the maze in real-time:
- **BFS (Breadth-First Search)**: Guarantees the shortest path by exploring layer by layer.
- **DFS (Depth-First Search)**: Explores deep paths aggressively (snake-like movement).
- **Quantum Search**: A simulated probabilistic search inspired by Grover's algorithm.

## Tech Stack

- **Frontend**: React, Vite
- **3D Graphics**: Three.js, React Three Fiber, Drei
- **Styling**: Tailwind CSS, Glassmorphism UI
- **Icons**: Lucide React

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```
4.  **Build for production**:
    ```bash
    npm run build
    ```

## Controls

- **Regenerate Universe**: Create a new random maze.
- **Start BFS / DFS / Quantum**: Visualize the selected algorithm.
- **Entropy Toggle**: Turn on/off dynamic wall shifting.
- **Simulation Speed**: Adjust how fast the algorithms run.
